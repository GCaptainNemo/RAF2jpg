# RAF2JPEG

该项目旨在将Fuji相机原始数据格式(.RAF)转换成(.jpg)格式。

# 使用方法

```python main.py -i RAF_INPUT_DIR -o JPEG_OUPUT_DIR```

Demo : ```python main.py -i ./img/ -o ./img/out/```
